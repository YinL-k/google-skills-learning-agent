# Autonomy Policy

The MVP favors guarded automation over full unattended operation.

- Course reading and state extraction are automatic.
- Navigation is automatic only when the visible control is known safe.
- Cloud Shell command execution is allowed only for tutorial-visible commands after lab account/project confidence is established.
- Ambiguous state becomes a hard stop.
- Human action is required for login, credentials, personal account choices, quizzes, assessments, billing, security checks, and End Lab.
